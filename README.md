## Build & Run

Requires <a href="https://nodejs.org/" target="_blank" rel="noopener noreferrer">Node.js</a>

```bash
# 1. Clone repo
git clone https://github.com/priyanshurav/libreludo.git
cd libreludo

# 2. Install dependencies
npm install

# 3. Start dev server
npm run dev

# 4. Build for production (Optional)
npm run build
```
